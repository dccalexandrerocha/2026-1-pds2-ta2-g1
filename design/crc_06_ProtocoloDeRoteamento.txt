================================================================================
  CLASSE: ProtocoloDeRoteamento             CATEGORIA: Estrategia (Classe Base)
================================================================================

  [US relacionadas: US03, US04]

  Nota: ProtocoloDeRoteamento e uma classe base abstrata que define a interface
  comum para algoritmos de roteamento. A subclasse concreta implementada neste
  projeto e:
    - RoteamentoDijkstra : calcula o caminho mais curto pelo algoritmo de
                           Dijkstra, usando os pesos (latencia) dos enlaces.

  Novas variações de algoritmo podem ser adicionadas no futuro como novas
  subclasses, sem modificar o restante do sistema (princípio Open/Closed).
